Detailed Description

This struct holds the date time data for the system. This structure is what is used everywhere but the file system when it comes to dates and times.

Parameters
inttm_sec - Seconds after the minute - [0, 60] including leap second.
inttm_min - Minutes after the hour - [0, 59].
inttm_hour - Hours since midnight - [0, 23].
inttm_mday - Day of the month - [1, 31].
inttm_mon - Months since January - [0, 11].
inttm_year - Years since 1900.
inttm_wday - Days since Sunday - [0, 6].
inttm_yday - Days since January 1 - [0, 365].
inttm_isdst - Is daylight savings time. This is unsupported.
