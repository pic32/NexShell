FRESULT GetDiskSpace ( BYTE * Drive,
DISK_SPACE_TYPE DiskSpaceType,
DISK_SPACE_UNITS DiskSpaceUnits,
UINT64 * Size )

Gets the disk space of the specified drive down to the nearest 1,024 bytes.

Parameters
[in]char*Directory - The volume drive to get the space of. This is something like "R:/".
[in]DISK_SPACE_TYPEDiskSpaceType - The type of disk space to get the value of.
[in]DISK_SPACE_UNITSDiskSpaceUnits - The units to get the space in.
[in,out]UINT64*Size - The calculated space is written here upon success of the method.
Returns
FRESULT - The result of the operation. If anything but FR_OK is returned, the value in *Size cannot be considered valid.

This function is called to find various types of space on a drive. The drive must have been initialized with a call to f_mount() prior to calling this method. This method will calculate the empty space down to the nearest 1,024 byte chunk. To get the exact space down to the byte use the method GetDiskSpacePrecise().
